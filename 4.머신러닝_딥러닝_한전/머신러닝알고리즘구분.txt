# scikit-learn : 파이썬 머신러닝 라이브러리

# 머신러닝의 응용분야
    - 분류(Classification)
       특정 데이터 레이블을 붙여 분류할 수 있다.
 
    - 클러스터링(Clustring)
        값의 유사성을 기반으로 데이터를 여러 그룹으로 나누는 것

    - 추천(Recommendation)
        특정 데이트를 기반으로 다른 데이터를 추천하는 것
        ex) 인터넷 서점에서 구매한 책들을 기반으로 다른 책을 추천하는 경우
    
    - 회귀(Regression)
        과거의 데이터를 기반으로 미래의 데이터를 예측하는 것
        ex) 판매 예측, 주가변동

    - 차원축수
        데이터의 특성을 유지하면서 데이터의 양을 줄여주는 것
